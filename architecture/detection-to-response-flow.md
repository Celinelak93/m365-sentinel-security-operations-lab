# Detection to Response Flow

## Purpose

This document explains how a suspicious event becomes a security incident and how the security team responds.

## Flow

```text
Security Event
  ↓
Detection Logic
  ↓
Analytics Rule
  ↓
Alert
  ↓
Incident
  ↓
Triage
  ↓
Investigation
  ↓
Containment
  ↓
Remediation
  ↓
Reporting
  ↓
Control Improvement
