# Microsoft 365 Security Operations & NIS2 Readiness Lab

This project demonstrates how to design a practical security operations capability for a small or medium-sized organization using Microsoft Defender XDR, Microsoft Sentinel, Entra ID, and Microsoft 365 security tools.

The lab focuses on four major security areas:

1. Identity security
2. Endpoint security
3. Email security
4. Cloud/SaaS security

The project includes:

- Security coverage inventory
- KQL detection queries
- Sentinel analytics rule templates
- Incident response playbooks
- NIS2/security governance mapping
- Executive security roadmap
- Synthetic sample data for demonstration
